# 🛡️ Thikana Dhabla Ghosi - Badgujar Rajput Heritage Website

![Website Preview](1200.jpeg)

## 📖 Overview

**Thikana Dhabla Ghosi** is a royal heritage website dedicated to preserving and showcasing the proud legacy of the **Badgujar Rajput family** from Thikana Dhabla Ghosi, Shajapur, Madhya Pradesh. This digital archive brings to life the valor, traditions, and genealogy of one of India's distinguished Suryavanshi warrior clans.

## 🏰 About the Project

This website documents the glorious history of the **Badgujar Rajputs** — a noble Suryavanshi clan tracing their divine lineage to **Lava**, the elder son of **Bhagwan Shri Ram**. The platform serves as both a historical record and a modern digital tribute to the royal family's enduring legacy.

### ✨ Key Features

- **Royal History** – Detailed chronicles of the Badgujar Rajput dynasty and the Thikana Dhabla Ghosi estate  
- **Family Tree** – Complete genealogical record from **Raja Goga Dev** to the present generation  
- **Interactive Map** – Location guide with Google Maps integration  
- **Contact Form** – Secure, CAPTCHA-protected communication with the family  
- **Privacy Policy** – Transparent data protection information  
- **Sitemap** – Complete site navigation structure for easy access
